# Compliance Map
This document maps repository artifacts to common frameworks:
- OpenSSF Scorecards: workflows (scorecard.yml), dependencies (dependabot), CI protections.
- CII Best Practices: docs (SECURITY, CONTRIBUTING, CODE_OF_CONDUCT), governance, tests.
- SPDX/REUSE: SPDX headers + REUSE.toml; SBOM spdx-json via workflow.
- SLSA: provenance example and release signing templates.
